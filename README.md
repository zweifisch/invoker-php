
php backend for [invoker](https://github.com/zweifisch/invoker)

install using [composer](http://getcomposer.org/)

```javascript
{
  "require": {
    "zweifisch/invoker": "*"
  }
}
```

usage see [invoker](https://github.com/zweifisch/invoker)

cd test
